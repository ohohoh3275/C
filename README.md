# C Language

---

### to complie .c file

1. check
   `gcc -v`

2. make file for compiling
   `gcc pointer_in_array.c -o thisisrunning`
   (naming 'thisisrunning' is for convenience, usually goes with .c-file's name)

3. run (2)
   `./thisisrunning`
